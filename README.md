# addition
This is a script made with C++ for addition

# How to install it:

git clone https://github.com/nektarioskouro/addition.git

cd addition

g++ -o addition addition.cpp

# How to run it:

cd addition

./addition

if you get any erros during the installation, just type : ./addition
